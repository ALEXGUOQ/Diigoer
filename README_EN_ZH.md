## Diigoer
Digoer 是一个Diigo Android 客户端，实现了登录，查看书签，对书签的分类，查找，分享（也可分享其他URL 到Diigo），缓存，设置主题等功能：
- [http://www.github.com/Tikitoo/Diigoer]()

## 安装此软件
- [diigoer_app_0.1.0_02.apk](http://7u2jvr.com1.z0.glb.clouddn.com/diigo-add-mark.png)

## 运行环境
- Android 4.0+
- Android Studio 1.02
- Gradle 2.2.1
